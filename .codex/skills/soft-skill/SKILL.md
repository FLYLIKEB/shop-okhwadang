# Soft Skill

Use this skill when the user wants an expensive, refined, agency-style visual treatment.

## Core direction

- Avoid default system-web typography such as `Inter`, `Roboto`, `Arial`, `Helvetica`.
- Use premium spacing with large section rhythm.
- Prefer softened structure, subtle depth, and restrained shadows.
- Use custom easing instead of generic `linear` or `ease-in-out`.
- Keep mobile layouts simple and collapsed; avoid `h-screen`, prefer `min-h-[100dvh]`.

## Components

- Favor nested surfaces, soft outlines, and deliberate CTAs.
- Motion should feel smooth and physical, not flashy.
- Keep blur and grain effects constrained and performance-safe.
