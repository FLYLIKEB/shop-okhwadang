# Brutalist Skill

Use this skill when the user wants a raw industrial interface rather than a polished consumer UI.

## Visual directions

Pick one and commit:
- Swiss industrial print: off-white substrate, hard grid, bold uppercase typography, red structural accents
- Tactical terminal: dark telemetry panels, monospace data density, CRT-like framing

## Rules

- Strong geometry, visible dividers, and sharp corners.
- Heavy type for macro structure, monospace for data.
- Avoid soft consumer-card styling.
- Use semantic technical UI where appropriate: `data`, `samp`, `kbd`, `output`, `dl`.
- Keep the brutality intentional; the UI still has to be readable and operable.
