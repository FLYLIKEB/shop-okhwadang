# Minimalist Skill

Use this skill when the user asks for a clean, editorial, Notion/Linear-like interface.

## Rules

- Favor monochrome or near-monochrome palettes with muted supporting tones.
- Use whitespace as the main visual separator.
- Prefer restrained radii, thin borders, and very soft shadows.
- No decorative clutter, gradients, neon effects, or loud color fields.
- Typography should be calm and highly legible.
- Use compact, intentional motion only where it improves scanability.

## Layout guidance

- Keep content width constrained.
- Establish hierarchy with spacing and type instead of ornament.
- Use subtle card elevation only when needed for grouping.
