# 서비스 기획서

상위 문서: [[README|옥화당 기획서 홈]]
관련 문서: [[feature-policy-map|기능별 정책 지도]], [[implementation-priority|구현 우선순위]]

## 제품 개요

옥화당 자사몰은 자사호, 보이차, 다구 전문 브랜드의 독립 D2C 쇼핑몰이다.
스마트스토어 의존도를 줄이고 브랜드 스토리텔링, 고객 데이터 소유, 글로벌 판매 기반을 확보하는 것이 목적이다.

## 핵심 목표

| 목표 | 설명 |
|------|------|
| 브랜드 스토리텔링 | 작가, 산지, 흙, 제작 과정 등 상품 가치를 콘텐츠로 전달 |
| 독립 구매 플로우 | 상품 탐색부터 장바구니, 주문, 결제, 배송 조회까지 자체 처리 |
| 운영자 편집성 | 어드민에서 상품, 주문, 배송, 페이지 블록, 네비게이션을 관리 |
| 글로벌 확장 | 한국어/영어, KRW/USD 결제, 해외 배송을 고려한 구조 |
| 외부 연동 독립성 | 국내 PG, 글로벌 PG, 배송사, 스토리지 공급자를 어댑터/설정으로 교체 가능하게 설계 |

## 사용자 범위

| 사용자 | 주요 작업 |
|--------|-----------|
| 비회원 방문자 | 상품 탐색, 검색, 상세 확인, 장바구니 접근 |
| 회원 | 주문/결제, 주문 내역 확인, 배송지 관리, 위시리스트, 쿠폰 사용 |
| 관리자 | 상품/주문/배송/회원/쿠폰/페이지/네비게이션 관리 |
| 슈퍼 관리자 | 관리자 권한 포함, 고위험 운영 권한 담당 |

## 주요 화면 범위

### 공개 화면

| 경로 | 화면 | 설명 |
|------|------|------|
| `/` | 메인 | 히어로, 추천/인기 상품, 카테고리 바로가기 |
| `/products` | 상품 목록 | 검색, 카테고리 필터, 정렬 |
| `/products/:id` | 상품 상세 | 이미지, 작가/산지 스토리, 옵션, 리뷰 |
| `/search` | 검색 결과 | 검색어 기반 상품 목록 |
| `/cart` | 장바구니 | 상품 수량 변경, 합계 계산 |
| `/login` | 로그인 | 이메일, 카카오, 구글 로그인 |
| `/register` | 회원가입 | 이메일 회원가입 |
| `/notice`, `/faq`, `/event` | 콘텐츠 | 공지, FAQ, 이벤트 |

## 기술 방향

| 영역 | 기준 |
|------|------|
| 프론트엔드 | Next.js App Router, React, TypeScript, TailwindCSS, Radix UI |
| 백엔드 | NestJS, TypeORM, MySQL, Controller-Service-Entity 구조 |
| 인증 | JWT, OAuth, RBAC |
| 결제 | 국내: 네이버페이/KG이니시스/토스페이먼츠, 글로벌: Stripe. 서버 사이드 금액 검증 |
| 배송 | 배송사 어댑터 패턴, 운송장/상태 추적 |
| 스토리지 | S3 또는 호환 스토리지 추상화 |
| 배포 | Vercel 프론트엔드, EC2 백엔드, Lightsail MySQL |

## 문서 기준 한계

- 이 문서는 기존 문서 기준의 기획 정리이며, 실제 코드 구현 완료 여부는 별도 확인이 필요하다.
- 결제 운영 정책은 국내 네이버페이/KG이니시스/토스페이먼츠 전체 지원, 글로벌 Stripe 우선으로 확정했다. 현재 코드는 일부 어댑터 구현이 뒤따라야 한다.
- CMS형 블록 빌더, 글로벌 스토어, 외부 리뷰/인스타그램 연동은 기획 범위에 있으나 구현 상세는 추가 확인이 필요하다.
- 글로벌 언어 범위는 한국어/영어만 지원한다. 기존 코드에 남아 있는 일본어/중국어 필드는 후속 정리 대상이다.
