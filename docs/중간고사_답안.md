## 1. 소수 문학(minor literature) + 사키야마 다미

들뢰즈와 가타리가 정의한 '소수 문학(minor literature)'의 개념은 무엇이며, 이를 사키야마 다미의 오키나와 문학에는 어떻게 적용할 수 있는가?

**[소수 문학 개념]**
- 수적 소수 ❌ → **다수 언어 속 소수성이 만드는 문학** ✓
- 다수파: 자기동일성의 규범·척도를 **사전에 지님** → 지배 이데올로기의 산물
- 소수파: 자명한 척도 없음 → '어떤 것이다'라는 확정적 표명 거부
- 핵심: **언어의 탈영토화** = 지배 언어 안에서 그 언어 바깥으로 나가기
- '언어=영토=국가=민족'이라는 획일화된 경계에 **균열**
- 소수 문학 = 지배 언어를 내부로부터 **비틀고 이질화**하는 정치적 실천

**[사키야마 다미 적용]**
- 오키나와 역사: 류큐왕국 → 1872년 일본 강제 병합 → 오키나와 전투(1945) → 미군 점령 → 1972년 '복귀' → 현재 미군기지 집중 = **군사주의+자본의 신식민지적 착취 구조**
- 실천: 일본어로 쓰되 **오키나와 방언(섬말)** 사용 → '접목된 언어' → 지배 언어를 내부에서 변형
- 〈보이지 않는 거리에서 숀카네가〉: 기지촌 유흥업소 = 군사주의+성적 착취가 결합된 자본주의 공간
- "가르쳐주겠다" 장면 → '치유의 남쪽 섬' 오키나와 표상(관광 자본의 이데올로기) **전복**
- 피착취·소외된 자의 위치에서 현실을 다시 언어화 = **소수 문학의 실천**

## 2. 박수남의 기록 작업
2. 박수남 감독의 조선인 원폭 피해자 기록(글쓰기 및 다큐멘터리 작업)의 과정은 어떠했으며, 그 기록들이 전하는 메시지는 무엇인가?

**[궤적]**
1. **1958년 고마쓰가와 사건**: 재일조선인 2세 이진우 → 3개월 만에 사형 선고 → 일본 사회의 구조적 인종주의·계급적 편견이 사법에 작동
2. **『죄와 죽음과 사랑』(1963)**: 이진우와 서신 교환 → 출간 → 총련에서 '반국가 인물' 낙인 → 남한·북한·일본 **어디에도 속하지 못함** → 국가·이념 상부구조 모두에 포섭되지 않는 경계적 주체 자각
3. **이진우 → '또 하나의 R'들**: 김희로·양정명·서승·서준식 → '일본인도 조선인도 아닌' 이중 정체성 → 소수자 안의 소수자
4. **1965년~**: 히로시마·나가사키 **조선인 원폭 피해자 증언** 기록 착수
5. **『또 하나의 히로시마: 조선인 한국인 피폭자의 증언』(1982)**: 증언 기록집
6. **〈또 하나의 히로시마: 아리랑의 노래〉(1986)**: 다큐멘터리
7. **〈되살아나는 목소리〉(2023)**: 딸 박마의와 미공개 필름 복원 → 세대 간 기억 계승

**[핵심 비판]**
- 히로시마 '부흥·평화도시' 서사 = **국가권력+자본 주도 이데올로기** → 조선인 피폭자 비가시화
- "육체노동으론 뼈 못 찾는데, 개발 땅에선 뼈가 나오더라" → 자본의 논리가 희생자 역사를 **문자 그대로 매장**
- 교차적 억압: "여자는 남자 아래, 조선인은 인간 아래" → 민족·젠더·계급·장애 **중첩**
- **느린 폭력(slow violence)**: 순간적 파괴 → 시공간 넘어 지속되는 고통 → 피해가 특정 계급·젠더·민족에 **불균등하게 전가**

**[메시지]**
- 폭력의 피해는 '보편 인류의 비극'으로 균질화 불가 → 물질적 조건에 따라 차별적으로 분배됨
- 연대는 소수자 안의 또 다른 소수자를 생산하는 **억압의 연쇄 구조**를 외면한 채 불가능

## 3. 표상의 정치 + 〈의형제〉
3. 에드워드 사이드가 제시한 '표상에 의한 지배' 혹은 '표상의 정치'란 무엇이며, 이를 통해 영화 〈의형제〉에 나타난 타자 표상을 어떻게 이해할 수 있는가?

**[사이드의 표상 개념]**
- 표상 = 현실 반영 ❌ → **이데올로기의 작용에 따라 구성**된 것 ✓
- 표상의 정치: 지배 이데올로기가 문학·영화·이미지를 통해 **자연화·재생산**되는 메커니즘
- 오리엔탈리즘 = 제국주의적 물질 조건 위에서 작동하는 **지식-권력** 장치
- 사이드의 표상 = "지배·통제 목적으로 주체가 객체를 **왜곡되게 표현**하는 행위"
- 타자의 표상 = '나(우리)' 정체성의 **허구성·불안정성**을 감추기 위한 신화적 이미지

**[〈의형제〉 분석]**
- 표면: 탈북민+전직 국정원 요원의 연대 → 남북 화합
- 실제 구조: '한민족 남성' 형제애 구성을 위해 **동남아 이주 외국인·결혼이주 여성·혼혈아**를 제3의 타자로 배치·후경화·대상화
- 이주 노동자·결혼이주 여성 = 한국 자본주의 노동력 재생산 구조에서 **가장 착취에 취약한 계급적 위치**
- 이들을 서사의 배경으로 **물화(物化, reification)** → 계급적 소외를 민족주의 이데올로기로 **봉합**
- 결론: 계급 모순을 은폐하고 지배적 민족주의 이데올로기를 **자연화**하는 이데올로기적 기능

## 4. 염상섭 소설의 일본인 표상
4. 염상섭의 소설 『이심』, 『사랑과 죄』, 『무화과』에서 일본인(식민자)의 표상이 지니는 의미는 무엇인가?

**[전제: 식민자 표상의 불가능성]**
- 강경애 『인간문제』: 일본인 = "게다 소리" → 신체 일부·사물로만 제시
- 원인: 검열 + 피식민자의 무의식적 **공간 분할 감각** = 식민지적 생산관계가 의식을 구조적으로 규정

**[염상섭의 특이성: '밀실']**
- 호텔 객실·카페·아틀리에 = 자본주의적 소비·유흥의 밀실
- 식민자+피식민자 대면의 **유일한 합법적 장소** = 식민 자본이 일상에 침투하는 물질적 접점
- **핵심 구조**: 조선인 사회주의자 퇴장 → 일본인 식민자 등장 (연동)

**[세 소설]**

| 소설 | 일본인 | 핵심 구조 | 의미 |
|---|---|---|---|
| 『이심』(1928) | 좌야 평일랑 (호텔 지배인) | 창호(사회주의자) 투옥 → 좌야 등장 → 춘경 착취·유곽 전락·자살 | 인간의 **상품화·물화** / 계급 투쟁 탈정치화 |
| 『사랑과 죄』(1927) | 심초매부 ('일한합병의 숨은 공로자') | 사회주의자 호연 퇴장 → 심초매부 후원자로 교체 | 저항 주체 소멸 → 식민 이데올로기 심화 (변증법적 구조) |
| 『무화과』(1931~32) | 안달외사 ('경성 낭인', 밀정) | 사회주의자 전무 → 안달외사가 정치적 진공 잠식 | 변혁 가능성 봉쇄 / **무화과(無花果)** = 계급 투쟁의 꽃 없이 자본의 열매만 |

**[결론]**
- 일본인 표상 = 식민지 자본주의의 일상 침투 + **계급적 저항 주체의 패배** 음각
- '무엇이 표상되었는가' ↔ '무엇이 표상되지 못했는가' (연동)

## 5. 루쉰의 시각적 조우(visual encounter)
5. 레이 초우가 『원시적 열정』에서 언급한 루쉰의 '시각적 조우(visual encounter)'란 무엇인가?

**[사건]**
- 루쉰, 도쿄 유학 중 강의에서 뉴스 영화 관람
- 내용: 러일전쟁 → 러시아 협력 중국인 처형 장면 + 무표정하게 방관하는 중국인 포로들

**[두 가지 충격]**
1. 처형당하는 자 + 방관하는 자 = 제국주의 앞에서 **소외된 개별 원자들로 분열**된 중국 민중의 상태 가시화 → '국민'으로서의 집합적 주체 부재
2. 스크린을 통해 처형당하는 중국인·방관하는 중국인·방관자인 자기 자신을 **동시에** 포착 → 피식민지 민중의 소외된 조건을 '중국인'이라는 개념 속에서 의미화

**[의의]**
- 새로운 미디어(영화) = **물질적 토대의 변화(생산력)** → 이데올로기·의식의 변형을 촉발
- 루쉰: 의사 → 작가로 전환 = 신체 치료보다 **제국주의적 소외로 마비된 민중 의식**의 변혁이 근본 과제라는 인식

## 6. 서양 인류학·민족지 비판 + 자기 민족지
6. 레이 초우가 『원시적 열정』에서 서양 인류학과 민족지를 비판한 이유는 무엇이며, 이를 극복할 대안으로 제시한 것은 무엇인가?

**[비판: 인류학적 교착(anthropological deadlock)]**
- 서양 인류학의 구조: '문명'의 과학적 관찰 방식 → '원시' 세계 연구
- 문제: 서구 지식 개입 순간 타자 문화 **이미 변형** / 서양적 모델이 항상 보편적 '기준'
- 이 불평등은 우연 ❌ → **제국주의적 물질 조건** 위에서 성립한 지식-권력 체계
- 딜레마: 비서양을 인류학적으로 파악 안 하면 학계 발언 불가 / 파악하면 서양 범주로 왜곡 → **교착**

**[대안: 자기 민족지(autoethnography)]**
- 핵심 전환: **'보는 행위'** (관찰자의 시선) → **'보여지는 것'** (대상이 스스로 자기를 표상)
- 정의: 과거 민족지의 **대상이었던 자**가 스스로 저자가 되는 새로운 민족지
- 메리 루이스 프랫: "식민화된 주체가 식민자의 표현 양식에 **선택적으로 협력·전유**"
- 재일 조선인 문학 적용: 일본어(지배 언어)로 쓰되 비틀고 전유 → 식민자의 조선인 표상을 내부에서 해체 = **자기 민족지**

## 7. 벤야민 + 레이 초우의 번역론
7. 발터 벤야민과 레이 초우의 '번역'에 대한 관점을 통해 우리는 문화의 형성과 변용을 이해하는 데 어떤 가능성을 얻을 수 있는가?

**[벤야민의 번역론]**
- 번역 = 의미 이전 ❌ → **언어 자체를 자기 재귀적으로 문제화**하는 행위
- 모든 언어 속에 잠재된 **'순수 언어'** → 번역을 통해서만 드러남
- 번역 = "언어의 **썩은 울타리**를 파괴하는 것" = 민족·국가에 폐색된 언어의 한계 돌파
- 시대 맥락: 파시즘이 언어를 국가 권력 도구로 징발하던 시기 → 번역론 = **이데올로기 비판**
- 『기술복제시대의 예술작품』: 복제 → 아우라 상실 → '숭배 가치'에서 **'전시 가치'**로 → 대중문화의 정치적 가능성

**[레이 초우의 문화번역]**
- 문화번역 = 언어 간 이동 ❌ → **서로 다른 문화·계급 문화·기호 체계 사이의 교섭·경합** 전체
- 이 과정은 중립 ❌ → 전문 지식·국가 권력·자본이라는 **권력의 장** 속에 항상 놓여 있음
- 대중문화의 '표층성' = 결여 ❌ → 지배적 의미 체계를 흔드는 **전달 가능성**의 형식
- 문화번역 = **문화적 저항**

**[종합: 가능성]**
1. 문화 = 단일한 기원의 순수 실체 ❌ → **지배와 저항의 변증법적 산물**
2. 번역 = 헤게모니에 포섭되지 않는 **잠재적 가능성**을 드러내는 계기
3. 식민지 문학·재일 조선인 문학 = 지배 언어 체계를 내부에서 흔드는 **번역 행위이자 문화적 저항**
4. 문화 형성·변용 = 계급·권력의 물질적 모순이 언어·표상의 층위에서 관철되고 **저항하는 역동적 과정**

## 8. 한국사회의 동아시아 담론 + 분단·통일
8. 한국사회에서 '동아시아'는 어떻게 이야기되고 논의되어 왔는가? 특히 '분단'과 '통일'의 관점에서 그것이 논의되어 온 것에 대해 어떻게 생각하는가?

**[담론의 역사적 전개]**

| 시기 | 내용 | 특징 |
|---|---|---|
| 1990년대 초 | 탈냉전 → 동아시아론 등장 / 최원식·창비 그룹 | 서구 중심 지식 비판, 변혁 이론, 분단 체제론과 결합 |
| 1990년대 중반 | 확산기 / 오리엔탈리즘 비판·탈근대·탈식민주의와 결합 | 맑스주의 쇠퇴 후 이론적 공백을 채우는 '담론장' |

**[분단 체제론]**
- 분단 = 이념 대립 ❌ → **지구적·지역적·일국적 모순이 응축된 재생산 체제**
- 분단이 지배 계급 이해관계에 봉사하며 **민중에게 억압·착취의 비용 전가**
- 분단 체제론 = 궁극적으로 **분단 극복론이자 변혁론**

**[한계: 분단 체제론의 내적 모순]**
- '인간적 주체성' 회복 → 실은 **민족적 주체성** 내포 → 민족주의 넘으려다 민족 전제하는 아이러니
- 동아시아 내부의 위계 구조 외면 위험: 한국/일본/중국 ↔ 오키나와·대만·재일조선인·이주 노동자

**[백영서의 '이중적 주변의 시각']**
- 세계사적 차원: 동아시아 자체가 주변부 → 주변으로서 자신 인식
- 지역 내부: 한층 더 주변적인 존재(오키나와·북한·대만 등)에 대한 자신의 **연루** 자각
- **핵심 현장**: 진먼다오·오키나와·개성 = 중층적 모순이 응축된 공간

**[나의 입장]**
- 통일 담론이 진정한 변혁 기획이 되려면:
  - 자본·군사주의·국가권력이 착취·소외시킨 **모든 주변 주체들의 물질적 조건**을 중심에 둘 것
  - 그렇지 않은 통일 담론 = **지배 계급의 이해관계를 민족 전체의 이익으로 포장**하는 이데올로기적 기능
  - 월러스틴의 '소통적 보편성' 지향: 단일한 보편 ❌ → **복수의 주변들이 분유(分有)하는 보편성** ✓
