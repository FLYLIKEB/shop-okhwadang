## 2. 플랫폼 vs 자사몰 비용 구조 비교

| 구분 | 스마트스토어 / 임대형 플랫폼 | 직접 구축 자사몰 |
|------|--------------------------|----------------|
| **결제 수수료** | 2.0~3.5% (플랫폼 정책) | 1.5~2.5% (PG 직접 계약) |
| **월 고정비** | 5~15만 원 (유료 앱/플러그인 포함) | 약 15만 원 (서버 실비) |
| **기능 추가** | 매월 추가 결제 또는 구현 불가 | **0원** (소유 코드이므로 자유 확장) |
| **데이터 활용** | 제한적 (플랫폼 소유) | **무한대** (마케팅·재구매 유도 활용) |
| **디자인 자유도** | 템플릿 틀 안에서만 | 브랜드 아이덴티티 100% 반영 |
| **브랜드 스토리텔링** | 작가 소개·산지 이야기·자사호 관리법 등 깊이 있는 콘텐츠 전달 어려움 | **무한한 콘텐츠 자유도** — 산지별 작가 스토리, 자사호 고르는 법, 관리 가이드, 다도 문화 콘텐츠 등 자유롭게 구성 |
| **소유권** | 없음 (플랫폼 종료 시 데이터 손실) | **소스코드·DB·서버 전체 소유** |

## 4. 포함된 기술적 가치

1. **UI/UX 설계**
   - 디자인 전공자가 직접 설계한 UI/UX. 업계 표준 컴포넌트 라이브러리(Radix UI) 기반.

2. **기술 부채 제로 (Clean Architecture)**
   - 업계 표준 구조(Next.js + NestJS)로 작성. 다른 개발자가 즉시 이어받을 수 있는 코드.

3. **서버 비용 최적화**
   - 최소한의 AWS 비용으로 최대 퍼포먼스. Vercel Pro 플랜으로 안정적인 프론트엔드 서빙 (상업적 사용 필수).

4. **검색 엔진 구조 최적화 (SSR + JSON-LD)**
   - 구글·네이버 검색 상단 노출 유도. '자사호 추천', '의흥 자사호', '작가 자사호' 등 키워드로 유입 가능.
