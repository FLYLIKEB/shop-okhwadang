## 1. 결제 (PG사)

### 1-2. 결제 수단별 추가 비용

> 결제 수단마다 수수료 구조가 다릅니다. 간편결제(카카오페이·네이버페이)는 편의성이 높아 전환율에 유리하지만 수수료가 별도로 발생합니다. 고객이 주로 사용하는 결제 수단을 파악하면 전체 수수료 부담을 최소화할 수 있습니다.

| 결제 수단      | 수수료 / 비용          | 비고                     |
| ---------- | ----------------- | ---------------------- |
| 신용카드 (일시불) | 1.5~3.0%          | PG사·카드사 협약에 따라 상이      |
| 신용카드 (할부)  | 2.0~3.5%          | 무이자할부 시 가맹점 부담 +0.5~1% |
| 무통장입금      | 없음                | 수동 확인 필요               |
| 가상계좌       | 건당 ₩300~500       | 발급 건수만큼 과금             |
| 실시간 계좌이체   | 0.5~1.5%          | 은행 수수료 포함              |
| 카카오페이      | 1.5~2.5%          | 별도 가맹점 계약              |
| 네이버페이      | 2.0~3.5%          | 네이버쇼핑 연동 시 추가 수수료      |
| 토스페이       | 1.5~2.5%          | 토스페이먼츠 가맹 시 포함         |
| 페이팔        | 3.4~4.4% + $0.3/건 | 해외 결제 전용               |

### 1-3. 부가 서비스 비용

> 기본 결제 외에 필요할 수 있는 추가 서비스들입니다. 정기결제(구독)나 해외 결제는 처음부터 설계에 반영해야 나중에 추가 개발 비용이 발생하지 않으므로, 향후 계획까지 미리 확인해두는 것이 중요합니다.

| 서비스 | 비용 | 비고 |
|--------|------|------|
| 정기결제(빌링키) | 카드 수수료 동일 | 별도 API 개발 필요 |
| 에스크로 | 무료~건당 ₩100 내외 | PG사마다 상이 |
| 해외 결제(외화) | +1~2% (환전 스프레드) | 페이팔·엑심베이 등 |
| SMS 본인인증 | 건당 ₩35~55 | 가입 시 필요한 경우 |
| 정산 분리(마켓플레이스) | 별도 계약 | 셀러 분리 정산 필요 시 |

## 4. 물류 / 배송

> 배송 방식과 정책은 쇼핑몰 운영의 핵심 중 하나입니다. 자사호·보이차처럼 파손 위험이 있는 상품은 포장·배송 방식 결정이 특히 중요합니다. 배송비 정책(무료 기준금액, 지역별 추가비)은 고객 구매 전환율에 직접 영향을 미치므로 경쟁 쇼핑몰을 참고해 설정하시길 권장합니다.

### 4-1. 배송 방식

> 초기에는 직접 배송(자체 포장·발송)이 일반적이며, 물량이 늘면 3PL(물류 대행)로 전환하는 경우가 많습니다. 현재 운영 방식과 향후 계획을 함께 알려주시면 시스템 설계에 반영할 수 있습니다.

- [ ] 직접 배송 (자체 창고 운영)
- [ ] 3PL 위탁 (물류 대행사에 위탁 — CJ대한통운 e-풀필먼트, 팬다코리아 등)
- [ ] 풀필먼트 서비스 이용 (쿠팡 로켓, 네이버 풀필먼트 등)
- [ ] 드롭쉬핑 (재고 없이 공급사가 직접 발송)
- [ ] 디지털 상품 (배송 불필요)

### 4-2. 택배사 (복수 선택 가능)

> 택배사 선택은 배송 추적 API 연동과 직결됩니다. CJ대한통운은 API 연동이 가장 용이하고 점유율이 높아 일반적으로 권장됩니다. 현재 계약 중인 택배사가 있으면 그대로 유지할 수 있습니다.

| 택배사 | 특징 | 선택 |
|--------|------|------|
| CJ대한통운 | 점유율 1위, API 연동 용이 | ☐ |
| 한진택배 | 전국망 안정적 | ☐ |
| 로젠택배 | 중소몰 가격 경쟁력 | ☐ |
| 우체국택배 | 도서산간 강세 | ☐ |
| 롯데택배 | 롯데 계열 연동 유리 | ☐ |
