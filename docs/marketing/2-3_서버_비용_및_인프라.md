## 서버 구성 한눈에 보기

```
고객 브라우저
  └→ Vercel Pro (프론트엔드 화면)
       └→ AWS EC2 (쇼핑몰 서버)
            ├→ AWS Lightsail DB (데이터베이스 — 자동 백업)
            └→ AWS S3 + CloudFront (상품 이미지 저장 및 고속 전송)
```

모든 인프라는 **고객사 명의 AWS 계정**으로 운영됩니다.
개발자가 없어도 AWS 콘솔에서 직접 현황 확인이 가능합니다.

## 이 구성을 선택한 이유

### Vercel Pro — 상업용 프론트엔드 호스팅
- Hobby(무료) 플랜은 약관상 **상업적 사용 금지** — 수익 발생 서비스는 Pro 플랜 필수입니다.
- 팀원 계정 추가, 1TB 대역폭, 서버리스 함수 60초 실행 등 운영에 필요한 기능이 포함됩니다.

### AWS EC2 t3.small — 업계 표준 서버
- 국내외 대부분의 개발사가 사용하는 표준 구조로, **향후 다른 개발자에게 유지보수를 맡길 때도 바로 인수인계 가능**합니다.
- 트래픽이 늘어나면 서버 사양을 버튼 하나로 업그레이드할 수 있습니다.
- 보안그룹 설정으로 허가된 연결만 허용 — 해킹 시도 차단 및 기업 보안 감사 대응이 가능합니다.

### AWS Lightsail DB — 자동 관리 데이터베이스
- 주문·고객 데이터를 **7일치 자동 백업**으로 보호합니다.
- 별도 관리 없이 AWS가 DB 서버를 직접 운영해 장애 위험을 줄입니다.
- 매출이 커지면 더 큰 DB로 무중단 업그레이드가 가능합니다.

### S3 + CloudFront — 이미지 전용 저장소
- 상품 이미지를 서버와 분리 저장해 **서버 교체·장애 시에도 이미지는 안전하게 보존**됩니다.
- CloudFront(CDN)를 통해 전국 어디서나 빠른 이미지 로딩을 제공합니다.
- 이미지가 늘어도 서버 부하 없이 용량을 무제한 확장할 수 있습니다.
