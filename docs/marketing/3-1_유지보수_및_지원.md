## 1. 초기 안정화 무상 케어 (Golden Month)

납품 후 **1개월간**은 초기 안정화를 위한 집중 케어 기간입니다. 별도의 비용 없이 아래 항목을 보장합니다.

- **결함 무상 수리:** 개발 과정에서 발견되지 못한 기능적 오류(Bug) 즉시 수정.
- **환경 적응 지원:** 어드민 사용법 안내 및 초기 데이터 설정 관련 기술 지원.
- **긴급 장애 대응:** 서버 다운 등 서비스 이용 불가 상황 발생 시 최우선 복구.

**무상 케어 기간 장애 대응 SLA**

| 등급 | 기준 | 확인 시간 | 해결 시간 |
|------|------|----------|----------|
| **긴급** | 서비스 중단, 결제 불가 | 2시간 이내 | 8시간 이내 |
| **일반** | 기능 오류, 데이터 이상 | 24시간 이내 | 72시간 이내 |
| **경미** | UI 오류, 텍스트 오류 | 다음 영업일 | 다음 정기 패치 |

| 항목 | **Standard (안정형)** | **Premium (성장형)** |
|------|---------------------|-------------------|
| **추천 대상** | 운영이 안정화된 초기 브랜드 | 활발한 마케팅과 업데이트가 필요한 브랜드 |

| **보안 업데이트** | 분기별 보안 패치 적용 | 매월 보안 및 라이브러리 업데이트 |
| **기술 상담** | 월 2회 (이메일/채팅) | 무제한 (전담 채널 운영) |

## 3. 서버 인프라 관리 및 실비 안내

서버는 **옥화당 전용 계정**으로 세팅되어 운영됩니다 (세팅법 별도 공유 예정). 투명한 비용 관리를 위해 고객 명의 **AWS(Amazon Web Services)** 계정으로 직접 청구됩니다.

- **월 관리비:** 15만원 (서버 실비 포함 — Vercel Pro + AWS EC2 + Lightsail DB + S3/CloudFront). **도메인 비용은 별도.**
- **서버 계정:** 옥화당 전용으로 세팅 (세팅법 별도 공유 예정).
- **인프라 최적화:** 유지보수 계약 시, 불필요한 자원 낭비를 막아 서버 비용을 최소화하는 기술적 튜닝을 지속적으로 수행합니다.
- **백업 정책:** 매일 자동 백업을 설정하여 예기치 못한 사고 발생 시에도 24시간 이내의 데이터로 복구가 가능하도록 관리합니다.

## 4. 추가 기능 고도화 (On-Demand)

유지보수 범위를 넘어서는 대규모 기능 추가(예: 라이브 커머스 연동, 멤버십 시스템 고도화 등)는 별도의 **단위 기능 견적**을 통해 진행됩니다.

- **표준 가이드 준수:** 표준 구조로 작성되어 다른 개발자도 유지보수 가능.
- **우선순위 배정:** 기존 유지보수 계약 고객에게는 신규 기능 개발 시 우선적인 일정 배정과 비용 혜택을 제공합니다.
