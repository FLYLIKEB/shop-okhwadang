# 기술 스택 및 아키텍처

사용 기술 스택과 아키텍처 구조를 정리한 문서입니다.

---

## 1. Core Technology Stack

가장 안정적이고 생태계가 활발한 기술을 선별하여, 추후 어떤 개발자가 이어받아도 즉시 유지보수가 가능합니다.

|**영역**|**기술 스택**|**비고**|
|---|---|---|
|**Frontend**|**Next.js 15 (App Router)**, React 19|최신 SSR 기술 적용으로 검색 엔진(SEO) 최적화|
|**Backend**|**NestJS**, TypeScript, TypeORM|엔터프라이즈급 확장성을 가진 모듈형 아키텍처|
|**Database**|**MySQL 8.0**, Redis|고성능 데이터 처리 및 고속 캐싱 시스템|
|**Styling**|**TailwindCSS v4**, Radix UI|브랜드 아이덴티티에 맞춘 유연한 디자인 시스템|
|**Auth**|**JWT**, OAuth 2.0|카카오/구글 간편 로그인 및 보안 인증|
|**Infra**|**Vercel** (Frontend), **AWS EC2** (Backend), **AWS Lightsail** (DB), Docker|프론트엔드는 Vercel CDN, 백엔드·DB는 AWS로 역할 분리|
|**Runtime**|**Node.js 22 (LTS)**|최신 런타임 환경으로 성능 최적화|

---

## 2. Architecture Highlights

단단한 설계를 통해 비즈니스의 변화에 유연하게 대응합니다.

- **어댑터 패턴(Adapter Pattern) 적용:** 토스페이먼츠 등 PG사나 택배사 API를 어드민에서 손쉽게 교체할 수 있도록 인터페이스화하여 설계했습니다. 특정 업체에 종속되지 않는 독립성을 제공합니다.

- **서버 사이드 렌더링(SSR):** 모든 상품 페이지는 서버에서 렌더링되어 구글, 네이버 등 검색 엔진에 즉시 노출됩니다.

- **이미지 최적화 인프라:** S3/R2 호환 스토리지를 사용하여 대용량 이미지를 빠르게 로딩하고 서버 부하를 최소화합니다.

## 3. Project Structure

체계적인 폴더 구조를 통해 대규모 기능 추가 시에도 코드 엉킴이 발생하지 않습니다.

```
commerce-project/
├── src/ (Frontend — Next.js 15 App Router)
│   ├── app/           # 페이지 및 레이아웃 (App Router)
│   ├── components/    # 재사용 UI 및 shadcn/ui 래퍼
│   ├── lib/           # API 클라이언트 및 전역 로직
│   └── contexts/      # Auth, Cart 상태 관리
├── backend/ (Backend — NestJS)
│   ├── src/
│   │   ├── modules/   # 기능별 모듈 (Auth, Order, Payment 등)
│   │   ├── database/  # TypeORM Migration 및 설정
│   │   └── common/    # 공통 가드, 파이프, 인터셉터
├── docs/              # API 명세서 및 운영 문서
└── docker-compose.yml # 인프라 통합 관리
```

## 4. Scalable API Design

표준 RESTful API 설계를 통해 모바일 앱 확장이나 외부 서비스 연동이 용이합니다.

- **Auth:** 이메일 및 소셜 로그인, 프로필 보안 관리

- **Products:** 필터링, 검색(Full-text), 카테고리 트리 구조 관리

- **Orders/Payments:** 주문 생성, 결제 준비/승인/취소의 트랜잭션 처리

- **Shipping:** 실시간 배송 추적 및 운송장 자동화 연동

- **Admin:** 대시보드 통계, 상품/주문/회원 통합 제어


---

## 5. Deployment & Reliability

검증된 배포 파이프라인을 통해 서비스 안정성을 확보합니다.

1. **Vercel Edge Network:** 프론트엔드 자원을 전 세계 어디서든 빠르게 제공합니다. Next.js Rewrites를 통해 백엔드 API를 안전하게 프록시합니다.

2. **AWS EC2 + Docker:** 백엔드 서비스를 컨테이너화하여 트래픽 증가 시 유연하게 서버를 확장할 수 있습니다.

3. **AWS Lightsail MySQL:** 관리형 DB로 자동 백업 및 안정적인 데이터 보관을 보장합니다.

4. **CI/CD 자동화:** 코드 수정 시 자동 테스트 및 빌드가 수행되어 휴먼 에러를 방지합니다.
