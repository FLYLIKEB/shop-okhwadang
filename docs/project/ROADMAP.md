## Phase별 구현 범위

### Phase 1 — MVP (W3)
- 인증 (회원가입 / 로그인 / JWT)
- 상품 목록 & 상세 (SSR, 검색, 카테고리 필터)
- 장바구니 (Context + API 연동)
- Header / Footer / 공통 레이아웃

### Phase 2 — Core (W4)
- 주문 & 결제 (PG 연동 — 국내 네이버페이/KG이니시스/토스페이먼츠 + 글로벌 Stripe)
- 주문 완료 / 주문 내역 조회
- 배송 조회 연동
- 마이페이지 (프로필, 배송지 관리)

### Phase 3 — 부가 기능 (W5)
- 상품 리뷰 & 평점
- 쿠폰 & 포인트 + 인플루언서 쿠폰 코드
- 카카오톡/SMS 메시지 발송 (어드민)
- 글로벌 스토어 설정 (한국어/영어, KRW/USD)
- 네이버 스마트스토어 리뷰 연동은 별도 이슈로 분리
- 인스타그램 쇼핑 연동
- 위시리스트
- 관리자 대시보드
- 공지사항 / FAQ / 1:1 문의

### Phase 4 — QA & 배포 (W6–W8)
- E2E 테스트 전 기능 케이스
- 모바일 반응형 / 크로스 브라우저 검수
- 성능 측정 (Lighthouse — Core Web Vitals)
- 스테이징 배포 → 고객 UAT
- 보안 강화 (Rate limiting, CORS, JWT, SQL Injection/XSS 방어)
- 최종 배포 (Vercel + EC2 t3.small + Lightsail MySQL)
- 모니터링 설정 + 납품 문서 정리

---

## 배포 인프라

```
클라이언트 → Vercel CDN (Next.js SSR)
          → Vercel Functions (api/proxy.ts) → AWS EC2 t3.small (NestJS :3000)
                                            → AWS Lightsail MySQL :3306
```

- **프론트엔드**: Vercel Pro (상업적 사용 필수)
- **백엔드**: AWS EC2 t3.small + PM2 + Nginx
- **DB**: AWS Lightsail MySQL (7일 자동 백업)
- **스토리지**: AWS S3 + CloudFront
- **CI/CD**: GitHub Actions → EC2 SSH 배포
