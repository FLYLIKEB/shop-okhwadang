# 옥화당 자사몰 — 서비스 기획서

옥화당(자사호·보이차·다구 전문 브랜드)의 D2C 자사몰. 스마트스토어에서 독립 쇼핑몰로 전환하여 브랜드 스토리텔링과 데이터 소유권을 확보한다.

---

## 프로젝트 개요

**옥화당 자사몰**은 네이버 스마트스토어(https://smartstore.naver.com/ockhwadang)에서 운영 중인 자사호·보이차·다구 전문 브랜드를 독립 쇼핑몰로 전환하는 프로젝트입니다.

> 기술 스택/구현 방향성은 [ChaLog](https://github.com/FLYLIKEB/ChaLog) 프로젝트와 통일합니다.

---

## 핵심 설계 원칙

| 원칙 | 설명 |
|------|------|
| **브랜드 스토리텔링** | 작가·산지·제작 과정 등 자사호의 가치를 전달하는 전용 콘텐츠 구조 |
| **글로벌 스토어** | 한국어/영어 다국어, KRW/USD 이중 가격 |
| **템플릿 기반** | 모든 UI 조형 요소는 하드코딩이 아닌 템플릿으로 관리. 어드민에서 블록 조립으로 페이지 구성 |
| **DB 드리븐 UI** | 카테고리, 사이드바, 네비게이션 등 프론트 구조를 DB로 관리하고 어드민에서 CRUD |
| **반응형 우선** | 모든 화면이 데스크톱/모바일에서 깨지지 않는 flex 기반 UI |
| **보안 우선** | 어드민, 인증, PG 연동 등 전 영역에 보안 고려 |
| **기술적 독립** | PG사·배송사를 어드민 설정만으로 교체 가능한 어댑터 패턴 |
| **재사용성** | 외주 프로젝트마다 포크하여 빠르게 커스터마이징 가능한 구조 |

## 주요 기능 목록

### 상품 탐색
- 메인 페이지 (히어로 배너, 추천/인기 상품, 카테고리 바로가기)
- 카테고리별 상품 목록 (자사호·보이차·다구세트·차도구 등 옥화당 특화 카테고리)
- 상품 상세 (이미지 갤러리, 작가·산지 스토리, 옵션, 리뷰, 문의 탭)
- 상품 검색 (자동완성, 최근/인기 검색어)

### 구매 프로세스
- 장바구니 (추가/삭제/수량 변경, 합계 계산)
- 주문/결제 (배송지 입력, PG 결제, 쿠폰/적립금 적용)
- 주문 완료 및 주문 내역 조회

### PG 결제
- 어댑터 패턴 기반 PG 추상화 (토스페이먼츠, KG이니시스 등)
- 네이버페이 필수 적용
- 개발: Mock PG / 운영: 실제 PG
- 서버 사이드 금액 검증 필수
- 결제 상태 머신 (대기 → 승인 → 취소/환불)

### 배송 시스템
- 배송 상태 관리 (결제완료 → 상품준비중 → 배송중 → 배송완료)
- 택배사 API 추상화 (CJ대한통운, 한진, 롯데 등 어댑터 패턴)
- 운송장 등록 및 실시간 배송 추적
- 해외 배송 지원

### 회원
- 회원가입 (이메일/비밀번호)
- 로그인/로그아웃 + OAuth (카카오/구글)
- 마이페이지 (주문내역, 회원정보, 배송지 관리)

### 커뮤니케이션
- 상품 리뷰 (별점, 포토 리뷰)
- 네이버 스마트스토어 리뷰 연동
- 위시리스트 (찜하기)
- 쿠폰/적립금 + 인플루언서 쿠폰 코드
- 카카오톡/SMS 알림 발송
- 공지사항, FAQ, 1:1 문의

### 글로벌 스토어
- 한국어/영어 다국어 지원
- 국가별 가격 개별 설정 (KRW/USD)
- 인스타그램 쇼핑 연동

### 어드민
- 페이지/템플릿 관리 (블록 구성, 설정값 편집)
- 네비게이션 관리 (GNB, 사이드바, 카테고리 CRUD)
- 상품/주문/배송/회원 관리
- 쿠폰/프로모션 관리 (인플루언서 코드 발급)
- 카카오톡/SMS 메시지 발송
- 국가별 가격 관리
- 매출/주문/방문 통계 대시보드 (recharts)

## 화면 구성 / 페이지 맵

### 공개 화면 (비로그인 접근 가능)

| 경로 | 화면명 | 설명 |
|------|--------|------|
| `/` | 메인 | 히어로 배너, 추천/인기 상품, 카테고리 바로가기 |
| `/products` | 상품 목록 | 카테고리별 리스트, 필터/정렬 |
| `/products/:id` | 상품 상세 | 이미지 갤러리, 작가·산지 스토리, 옵션, 리뷰, 문의 |
| `/search` | 검색 결과 | 검색어 기반 상품 목록 |
| `/cart` | 장바구니 | 상품 목록, 수량 변경, 합계 |
| `/login` | 로그인 | 이메일 / 카카오 / 구글 |
| `/register` | 회원가입 | 이메일 회원가입 |
| `/notice` | 공지사항 | 공지 리스트/상세 |
| `/faq` | FAQ | 카테고리별 아코디언 |
| `/event` | 이벤트 | 기획전, 타임세일 |

### 로그인 필요 화면

| 경로 | 화면명 | 설명 |
|------|--------|------|
| `/checkout` | 주문/결제 | 배송지, 결제, 쿠폰 적용 |
| `/order/complete` | 주문 완료 | 주문 확인 |

## 구현 우선순위

```
Phase 1 (MVP)     : 메인, 상품 목록/상세, 장바구니, 주문/결제(Mock PG) — 핵심 구매 플로우
Phase 2 (Core)    : 회원 시스템, 검색, 카테고리 필터, 마이페이지
Phase 3 (Payment) : 실제 PG 연동 (네이버페이 필수), 배송 시스템 구축
Phase 4 (Admin)   : 어드민 (상품/주문/배송/회원 관리)
Phase 5 (CMS)     : 템플릿 시스템, 동적 페이지/네비게이션 관리
Phase 6 (Polish)  : 리뷰, 위시리스트, 쿠폰, 글로벌 스토어, 외부 연동
Phase 7 (Ops)     : CI/CD, 모니터링, 테스트 커버리지
```
